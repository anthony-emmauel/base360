# CLAUDE.md

## Design Context

This project has an `impeccable`-managed design system. Before making UI changes, read:

- **[PRODUCT.md](PRODUCT.md)** — register (brand), users, brand personality ("confident, blunt, fast"), anti-references (avoid generic glossy SaaS/DTC marketing), design principles.
- **[DESIGN.md](DESIGN.md)** — visual system: "The Signal Thread" north star, color/type/elevation tokens, component specs, Do's and Don'ts.

Key rule: Thread Violet (`#6c47ff`) is spent rarely (CTA, thread motif, hot-lead tag only); everything else stays flat with hairline borders except the one product-panel shadow float.
